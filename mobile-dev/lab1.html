Введение в Kivy: Первое приложение

Лабораторная работа по изучению фреймворка Kivy для создания мобильных приложений на Python

Цель

  • Ознакомиться с фреймворком Kivy, понять его преимущества для создания мобильных приложений на Python.
  • Изучить базовые компоненты и структуру приложения на Kivy.
  • Реализовать простое приложение, чтобы освоить основы работы с виджетами.

Теория

Kivy — это Python-фреймворк для создания графических пользовательских интерфейсов (GUI), который поддерживает разработку кроссплатформенных приложений (Android, iOS, Windows, macOS, Linux).

  • Подходит для мобильных устройств: сенсорный ввод, жесты.
  • Использует OpenGL ES 2 для создания гибкой и красивой графики.
  • Возможность писать код один раз и запускать на разных платформах.

Как установить Kivy

  1. Убедитесь, что Python установлен (рекомендуется версия 3.8+).
  2. Установите библиотеку Kivy через pip:
    3. pip install kivy
  3. Проверьте, установилась ли библиотека, создав простой Python-файл:
    4. from kivy.app import App
print("Kivy установлен!")

Базовая структура проекта на Kivy

Структура проекта в Kivy обычно включает два компонента:

  • Главный Python-файл, который создаёт приложение и подключает логику.
  • kv-файл (необязательно), в котором описывается интерфейс приложения.

Пример структуры проекта:

project/
|-- main.py
|-- myapp.kv

Практическая часть

Шаг 1. Создайте проект

Создайте папку для работы с проектом, например, Lab1.

Шаг 2. Реализация "Привет, мир!"

Создайте файл main.py с кодом:

from kivy.app import App
from kivy.uix.label import Label  # Виджет для текста

class MyApp(App):
    def build(self):
        # Создаём текстовый виджет
        return Label(text="Привет, мир!")

# Запуск приложения
if __name__ == '__main__':
    MyApp().run()

Шаг 3. Добавьте кнопку для изменения текста

Расширьте приложение, добавив кнопку для изменения текста:

from kivy.app import App
from kivy.uix.button import Button  # Виджет кнопки
from kivy.uix.boxlayout import BoxLayout  # Для компоновки элементов
from kivy.uix.label import Label  # Виджет для текста

class MyApp(App):
    def build(self):
        # Основной компоновщик
        layout = BoxLayout(orientation='vertical')

        # Текстовый виджет
        self.label = Label(text="Привет, мир!")

        # Кнопка
        btn = Button(text="Нажми меня")
        btn.bind(on_press=self.change_text)

        # Добавляем виджеты в компоновщик
        layout.add_widget(self.label)
        layout.add_widget(btn)

        return layout

    def change_text(self, instance):
        self.label.text = "Привет, Kivy!"

# Запуск приложения
if __name__ == '__main__':
    MyApp().run()
