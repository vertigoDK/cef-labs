Конвертер валют на Kivy

Лабораторная работа по созданию приложения "Конвертер валют" с использованием фреймворка Kivy

Цель

  • Научиться использовать функции для организации логики приложения.
  • Ознакомиться с созданием простого конвертера валют с использованием статических данных для курса валют.

Теория

В Python функции играют важную роль в организации логики программы. Это блоки кода, которые выполняют конкретные действия и могут принимать параметры для обработки данных. Мы можем использовать их для обработки ввода, выполнения расчётов и обновления интерфейса.

Основы работы с функциями в Python

Функции в Python создаются с помощью ключевого слова def. Функции могут принимать аргументы, которые передаются при вызове, и возвращать значения, которые могут быть использованы в программе. Вот пример создания простой функции: 

def my_function():
    print("Привет, мир!")

В примере выше функция my_function не принимает аргументов и не возвращает значения, просто выполняет команду print.

Функции могут также принимать параметры, что позволяет им быть более гибкими. Например: 

def add(a, b):
    return a + b

В этом случае функция add принимает два аргумента a и b, а затем возвращает их сумму.

Как создать логику для конвертера валют

Для создания конвертера валют в Kivy мы можем использовать заранее заданные данные для курсов валют. Это называется статическими данными, поскольку они не меняются во время работы программы. Мы можем создать словарь, где ключами будут валюты, а значениями их курсы по отношению к базовой валюте (например, USD).

Пример статического словаря для курсов валют: 

EXCHANGE_RATES = {
    "USD": 1.0,  # Курс USD к самому себе
    "EUR": 0.93,  # Курс EUR к USD
    "GBP": 0.82,  # Курс GBP к USD
    "INR": 82.65,  # Курс INR к USD
    "JPY": 148.56,  # Курс JPY к USD
    "AUD": 1.53  # Курс AUD к USD
}

Этот словарь позволяет нам конвертировать валюту в USD и наоборот.

Работа с интерфейсом в Kivy

Kivy позволяет легко создавать графические интерфейсы для приложений с помощью виджетов. Для создания конвертера валют мы будем использовать такие виджеты, как TextInput для ввода суммы, DropDown для выбора валюты и Button для выполнения конвертации.

TextInput позволяет пользователю вводить текст, и в нашем случае это будет сумма, которую нужно конвертировать. DropDown даёт возможность выбрать валюту, а Button запускает конвертацию, когда на него нажимают.

Логика конвертации будет заключаться в том, чтобы взять введённую сумму, умножить её на курс выбранной валюты и отобразить результат на экране.

Работа с функциями и событиями

Мы будем использовать функции для обработки событий, таких как нажатие кнопок или выбор валюты. Например, при нажатии на кнопку конвертации будет вызываться функция, которая выполнит вычисления и обновит интерфейс.

Пример реализации конвертера валют

Ниже приведён пример кода для создания конвертера валют с использованием Kivy и статических данных:

from kivy.app import App
from kivy.uix.button import Button
from kivy.uix.textinput import TextInput
from kivy.uix.label import Label
from kivy.uix.dropdown import DropDown
from kivy.uix.boxlayout import BoxLayout

# Статичные курсы валют
EXCHANGE_RATES = {
    "USD": 1.0,
    "EUR": 0.93,
    "GBP": 0.82,
    "INR": 82.65,
    "JPY": 148.56,
    "AUD": 1.53
}

class CurrencyConverterApp(App):
    def build(self):
        # Основной layout
        self.layout = BoxLayout(orientation='vertical', padding=10, spacing=10)

        # Ввод суммы
        self.amount_input = TextInput(hint_text="Введите сумму", font_size=32, multiline=False)
        self.layout.add_widget(self.amount_input)

        # Создание выпадающего списка для валют
        self.dropdown = DropDown()
        self.currencies = list(EXCHANGE_RATES.keys())  # Список доступных валют
        self.selected_currency = "USD"  # Валюта по умолчанию

        for currency in self.currencies:
            btn = Button(text=currency, size_hint_y=None, height=44)
            btn.bind(on_release=self.select_currency)
            self.dropdown.add_widget(btn)

        self.currency_button = Button(text="Выберите валюту", size_hint_y=None, height=44)
        self.currency_button.bind(on_release=self.dropdown.open)
        self.layout.add_widget(self.currency_button)

        # Метка для отображения результата
        self.result_label = Label(text="Результат: ", font_size=32)
        self.layout.add_widget(self.result_label)

        # Кнопка для выполнения конвертации
        self.convert_button = Button(text="Конвертировать", font_size=32)
        self.convert_button.bind(on_press=self.convert_currency)
        self.layout.add_widget(self.convert_button)

        return self.layout

    def select_currency(self, instance):
        """Обработчик выбора валюты"""
        self.selected_currency = instance.text
        self.currency_button.text = f"Валюта: {self.selected_currency}"

    def convert_currency(self, instance):
        """Обработчик для конвертации валюты"""
        amount = self.amount_input.text
        if not amount.isdigit():
            self.result_label.text = "Ошибка: введите корректную сумму!"
            return

        amount = float(amount)
        rate = self.get_exchange_rate(self.selected_currency)
        if rate:
            result = amount * rate
            self.result_label.text = f"Результат: {result:.2f} {self.selected_currency}"
        else:
            self.result_label.text = "Ошибка: не удалось получить курс!"

    def get_exchange_rate(self, currency):
        """Получение курса валюты из статического словаря"""
        return EXCHANGE_RATES.get(currency, None)

# Запуск приложения
if __name__ == '__main__':
    CurrencyConverterApp().run()

В этом примере:

  • Создаётся интерфейс для ввода суммы и выбора валюты.
  • Кнопка "Конвертировать" выполняет расчёт и отображает результат.
  • Статические данные о курсах валют используются для выполнения конвертации.
